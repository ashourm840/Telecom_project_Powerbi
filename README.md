# Telecom_project_Powerbi
#Overview :
This is a Power BI Dashboard that analyzes the reasons of churn of the customers at that telecom company 
#Key Features:
* Dynamic Filters: Users can filter by Account length, Payment method, and Category.
* Tracking: Monitored churn rates, churn reasons, contract type and international calls.
* Processed using Power Query.
* using DAX.
